Fitness Tracker

This application allows you to track different fitness and health goals
by using a GUI to log into a database. To run this application you need
to have java 25 installed on your machine using this link:
https://www.oracle.com/java/technologies/downloads/

To launch the application simply unzip the folder and launch the file labeled FitnessTracker-1.0-jar-with-depencies.jar 
which is located inside of the FitnessTracker-1/target folder.
The data which gets tracked is workouts, sleep, water, and calories. You can
do this by both logging them and also setting goals for what you want to achieve.
